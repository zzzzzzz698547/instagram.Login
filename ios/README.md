# iOS IG Login Template

This folder contains a SwiftUI iPhone-style login template inspired by the Instagram sign-in screen.

Files:
- `IGLoginTemplate/IGLoginTemplateApp.swift`
- `IGLoginTemplate/InstagramLoginTemplateView.swift`
- `IGLoginTemplate/LoginRecordsStore.swift`

Use this as a UI template in Xcode. It includes a safe demo record preview only; it does not store or display plaintext passwords.
